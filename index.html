Multi-Tenant SaaS Platform
— 200k Users, 65% Faster APIs
Led backend architecture for a B2B SaaS serving 200k+ users across 800+ tenant organisations. Identified N+1 queries and missing DB indexes that were silently degrading API response times. Introduced a Redis cache-aside layer, rewrote critical ORM queries using select_related and bulk_create, and migrated background jobs to an async Celery + SQS pipeline. Deployed on AWS ECS with blue/green deploys — 18 months running at 99.98% uptime.
Django DRF PostgreSQL Redis Celery AWS ECS RDS ElastiCache
